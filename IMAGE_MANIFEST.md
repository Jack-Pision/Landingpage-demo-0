## Image Guidelines

### Technical Specifications:
- **Format:** PNG for product images (transparency not required but nice), JPG for story images
- **Color Space:** sRGB
- **Resolution:** 72-150 DPI for web (export at 2x for retina displays)
- **Compression:** Optimize for web (use TinyPNG or similar)

### Style Consistency:
1. **Lighting:** Soft, diffused studio lighting. No harsh shadows. No reflections on lenses.
2. **Color Palette:** 
   - Products: Black, dark brown, charcoal, silver, gunmetal
   - Backgrounds: White #FAFAFA, Light gray #F0F0F0, Medium gray #E8E8E8
   - Avoid: Bright colors, high saturation, busy patterns
3. **Mood:** Quiet luxury, understated elegance, masculine refinement
4. **No:** Visible logos, text overlays, watermarks, busy backgrounds, trendy styling

### After Generation:
1. Place all images in `/public/images/` folder
2. If using PNG, ensure file sizes are optimized (< 200KB each if possible)
3. The code currently references `.svg` extensions — update `data/products.ts` to use `.png` or `.jpg` as appropriate
